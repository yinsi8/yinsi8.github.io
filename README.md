## 研发笔记
### 流程记录
- 部分文档是GB18030编码，需要转换成utf-8否则下载时是乱码
  ```bash
  iconv -f GB18030 -t UTF-8 ~/input_path/file.txt > ~/output_path/file.txt
  ```
- 最好能写个批量detect encoding + conversion的script，大家给的txt都是不同encoding啊……
